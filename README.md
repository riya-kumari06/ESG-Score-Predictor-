# 🌍 ESG Score Predictor Using Machine Learning

> Predicting corporate sustainability performance using Ridge Regression
> Aligned with UN SDG 8 · SDG 13 · SDG 17

---

## 📌 Overview

ESG (Environmental, Social, and Governance) ratings determine how
sustainably a company operates. These scores influence billions of
dollars in investment decisions globally — yet they are produced by
closed, expensive agencies like MSCI and Sustainalytics, making them
inaccessible to most organizations.

This project builds an open, machine learning-powered ESG Score
Predictor that takes a company's financial and sustainability metrics
as input and predicts its exact ESG Overall Score (0–100) using
Ridge Regression — replicating real-world rating agency logic at
zero cost.

---

## 🎯 Key Objectives

- Predict exact ESG Overall Score using supervised regression
- Identify key financial and sustainability drivers of ESG performance
- Deploy an interactive web app for real-time ESG prediction
- Support green skilling aligned with UN SDG 8, SDG 13, and SDG 17

## 📊 Dataset

| Property | Details |
|---|---|
| Source | Kaggle — ESG & Financial Performance Dataset |
| Rows | 11,000 |
| Columns | 16 |
| Year Range | 2015 – 2025 |
| Companies | 1,000 unique |
| Industries | 9 (Technology, Finance, Energy, etc.) |
| Regions | 7 (North America, Europe, Asia, etc.) |
| Target Variable | ESG_Overall (continuous, range 6.3–98.8) |

**Dataset Link:**
https://www.kaggle.com/datasets/shriyashjagtap/esg-and-financial-performance-dataset

**Key Features:**
- Financial: Revenue, Profit Margin, Market Cap, Growth Rate
- ESG Sub-scores: Environmental, Social, Governance (0–100)
- Operational: Carbon Emissions, Water Usage, Energy Consumption
- Categorical: Industry, Region, Year

## 🤖 Model

| Property | Details |
|---|---|
| Primary Algorithm | Ridge Regression (L2 Regularization) |
| Why Ridge? | Handles multicollinearity in ESG features |
| Optimization | GridSearchCV — 10-Fold Cross Validation |
| Scaling | StandardScaler (fit on train only) |
| Evaluation Metrics | MAE, MSE, R², MAPE |

**Models Compared:**
Linear Regression · Ridge · Lasso · ElasticNet ·
Random Forest · Gradient Boosting

## 🚀 How to Run

### 1. Clone the repository
```bash
git clone https://github.com/your-username/ESG-Score-Predictor.git
cd ESG-Score-Predictor
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the notebook
Open `ESG_Score_Predictor.ipynb` in Jupyter Notebook or
Google Colab and run all cells top to bottom.
This will generate the `model_artifacts/` folder.

### 4. Launch the web app
```bash
streamlit run app.py
```

> ⚠️ Run the notebook first — the app requires the
> model_artifacts/ folder generated by Step 11.
